# モジュールグリッド

PDF図面にモジュール線（通り芯）を引いて、線入りPDFの保存・印刷ができるツールです。
ブラウザだけで動きます（インストール不要）。

## 使い方

1. ページを開く（この後の設定が済めば GitHub Pages のURL、または `index.html` をダブルクリック）
2. 図面PDFを開く（複数PDF・複数ページ対応。ファイル名から1F/2Fを自動判定）
3. 「**図面から自動計測**」で縮尺を合わせる（寸法数字の並びから実効縮尺を検出。A4縮小出力されたPDFも正しく合います）
4. 「線を追加」→2点クリックで基準線→引きたい側をクリックで平行複製（途中で間隔を変えれば683mmなどの変則スパンもOK）
5. 「線入りPDFで保存」または「印刷」（全図面をまとめて1つのPDFに）

詳しくはアプリ内サイドバー最下部の「使い方ガイド」をどうぞ。

## プライバシー

図面PDFはすべて **お使いのPCのブラウザの中だけ** で処理されます。
サーバーへのアップロードは一切ありません（このページはネット接続が無くても、一度開けば動作します）。
